# Desktop-Changer
A scraping tool built to get a random image from, "Work Hard Anwhere" website and set it as your desktop background.

## Getting Started

### Prerequisites

A computer with python installed. 

### How to Run

Open the terminal in the cloned folder. 
```
python3 script.py
```
